
#  Portfólio — Samuell Aguiar

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar meus projetos, experiências e habilidades de forma clara e profissional.

O site está publicado em: [samuellaguiar.vercel.app](https://samuellaguiar.vercel.app/)

---

##  Tecnologias Utilizadas

*  [Vite](https://vitejs.dev/) — build tool
*  [React](https://react.dev/) — biblioteca para interfaces
*  CSS Modules — estilização por componentes
*  [Vercel](https://vercel.com/) — deploy e hospedagem

---

##  Estrutura do Projeto

```
src/
 ├── components/
 │   ├── aboutMe/        # Seção "Sobre mim"
 │   ├── contact/        # Seção de contato
 │   ├── experience/     # Experiências profissionais
 │   ├── footer/         # Rodapé
 │   ├── home/           # Página inicial
 │   ├── navBar/         # Barra de navegação
 │   ├── projects/       # Projetos desenvolvidos
 │   └── skills/         # Habilidades técnicas
 ├── App.jsx             # Componente principal
 └── main.jsx            # Entrada da aplicação
```

---

##  Como Executar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/SamuellAguiar/Portfolio.git
   cd Portfolio
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Abra no navegador:
   ```
   http://localhost:5173
   ```
